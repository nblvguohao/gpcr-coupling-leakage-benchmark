# 引用刷新技能

## 触发方式
- 手动触发：输入 "/citation-refresh" 或 "刷新引用"

## 功能描述
当你插入新引用或修改正文引用后，自动执行以下操作：

## 执行流程
1. **扫描全文中所有引用**
   - 提取所有 [Author, Year] 或（作者，年份）格式的引用
   - 去重，生成引用列表

2. **对比参考文献列表**
   - 检查正文引用的每一条是否在参考文献列表中存在
   - 检查参考文献列表中的每一条是否在正文中被引用
   - 标记"孤立引用"（只在文中出现但不在列表中）和"死引用"（只在列表中但未被引用）

3. **排序与格式化**
   - 将参考文献列表按字母顺序重新排列
   - 检查格式一致性（作者名大小写、年份位置、标点符号）
   - 对格式不一致的条目标注并建议修正

4. **输出报告**
   ```
   ## 引用刷新报告
   ### 新增引用：3 条
   ### 孤立引用：1 条 → [需补全文献信息]
   ### 死引用：2 条 → [建议删除或补引用]
   ### 格式问题：4 处 → [见详情]
   ### 更新后的参考文献列表：（完整列表）
   ```

## 约束
- 不编造任何文献信息
- 对孤立引用，只标注提醒，不尝试猜测完整信息
