# 图表与表格规范

## 命名规则
- 图：按章编号，格式为 "图 X-Y 图的标题"
  - 例："图 3-2 实验流程图"
- 表：按章编号，格式为 "表 X-Y 表的标题"
  - 例："表 4-1 描述性统计结果"

## 表格规范
1. 使用三线表格式（顶线、栏目线、底线）
2. 表头必须包含变量名和单位
3. 显著性标注：\*p < 0.05, \*\*p < 0.01, \*\*\*p < 0.001
4. 样本量 N 必须在表注中说明
5. 所有缩写必须在表注中展开

## 图表规范
1. 坐标轴必须标注名称和单位
2. 配色方案：黑白友好（考虑打印效果），不使用红绿对比（色盲友好）
3. 误差线必须标注（标准误 SE 或 95% 置信区间）
4. 图例清晰可辨，置于图内不被遮挡的位置

## 正文引用规则
1. 所有图表必须在正文中被引用
2. 引用格式："如图 X-Y 所示" 或 "（见表 X-Y）"
3. 图表不能孤立出现——每个图表前后至少各有 3 句讨论
4. 图表标题本身应自明——读者只看图表和标题即可理解核心信息

## 注释规范
- 一般注释：用 "注：" 开头
- 统计注释：标注检验方法和显著性水平
- 来源注释：注明数据出处（若非原创）
