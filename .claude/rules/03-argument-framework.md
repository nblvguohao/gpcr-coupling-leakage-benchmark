# 论证框架规则

## 核心结构：PEEL 模型
每个学术段落必须遵循 PEEL 论证结构：

- **P (Point)**：段落首句给出核心观点
- **E (Evidence)**：引用文献或数据作为证据
- **E (Explanation)**：解释证据如何支持观点
- **L (Link)**：回扣论点，或过渡到下一段落

## 论证链要求
1. **一个段落一个论点**：不在一段内混杂多个并行论点
2. **证据等级**：
   - Level 1：元分析/系统综述
   - Level 2：大规模实证研究
   - Level 3：案例研究/小样本实验
   - 禁止：个人经验、博客文章作为唯一证据
3. **反证处理**：论证必须考虑对立观点，不选择性忽视反面证据
4. **论证闭环**：引言中提出的每个研究问题，必须在结论中得到回应

## 段落间逻辑关系标记
- 递进：使用"进一步而言"、"更为关键的是"
- 转折：使用"然而"、"与此相对"
- 因果：使用"由此导致"、"其原因在于"
- 并列：使用"与此同时"、"另一方面"

## 禁止的逻辑谬误
- 以偏概全（从个别案例推导普遍结论）
- 因果倒置（混淆相关与因果）
- 循环论证（用结论证明前提）
- 稻草人谬误（歪曲对立观点再反驳）
