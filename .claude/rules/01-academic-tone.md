# 学术语调规则

## 基本原则
- **客观性**：优先使用被动语态和无人称结构，避免"我"、"我们"作为主语
- **严谨性**：所有论断必须有依据，推测性内容必须使用"可能"、"似乎"、"在某种程度上"等软化词
- **克制性**：不过度承诺研究发现的意义，不使用"彻底改变"、"首次发现"等夸大表述

## 语气要求
1. **被动优先**："实验采用了..." 而非 "我们采用了..."
2. **数据驱动**：以数据/文献为出发点，不以个人感受为出发点
3. **分层陈述**：先给出结论，再展开论证，最后给出限定条件
4. **避免修辞问句**：不使用"难道不是吗？"等反问句式

## 章节对应的语气梯度
- 引言：中立偏引导性，明确研究动机
- 文献综述：批判性，突出争议和缺口
- 方法：技术性，精确、可复现
- 结果：中性，纯描述，不加解释
- 讨论：解释性，但克制
- 结论：总结性，适度展望

## 禁用的主观表述
- "令人惊讶的是" → 直接陈述发现
- "遗憾的是" → "需要指出的是"
- "幸运的是" → 删除，保持中性
- "这一发现意义重大" → "这一发现表明..."
