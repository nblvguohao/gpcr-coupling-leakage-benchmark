# 初始化大纲命令

## 触发词
/init-outline 或 /init

## 期望行为
当我输入 "/init <论文题目>" 时，执行以下流程：

1. 读取根目录 `CLAUDE.md` 和 `rules/01-academic-tone.md`
2. 基于题目，生成一份完整的三级大纲（章 → 节 → 小节）
3. 大纲结构遵循标准学术论文格式：
   - 第一章：引言（研究背景、问题陈述、研究意义、论文结构）
   - 第二章：文献综述（理论基础、国内外研究现状、研究缺口）
   - 第三章：研究方法（研究设计、数据来源、分析方法）
   - 第四章：结果与讨论
   - 第五章：结论与展望
4. 每一节附带 1-2 句说明该节的核心目标
5. 标注各章之间的逻辑依赖关系（如：第三章依赖第二章的理论框架）

## 输出格式
```
## 论文大纲：《题目》

### 第一章 引言
#### 1.1 研究背景
> 目标：阐明研究问题的现实与理论背景
...
```
