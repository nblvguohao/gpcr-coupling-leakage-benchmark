# 最终研究报告（中文版）

**标题：** 基于 ESM-2 嵌入与 curated 跨膜注释的拓扑感知 GPCR–G 蛋白偶联预测

---

## 第一部分：背景与动机

### 1.1 生物学背景

G 蛋白偶联受体（GPCRs）是人类基因组中最大的膜蛋白家族，约占已批准药物靶点的 34%。它们通过与异源三聚体 G 蛋白（四大主要家族：Gq、Gi、Gs 和 G12/13）偶联来激活下游信号传导。因此，预测给定 GPCR 与哪个 G 蛋白家族偶联，是药物设计和信号转导研究中的核心问题。

结构生物学已证实，A 类 GPCR 与 G 蛋白的相互作用界面主要位于胞内侧，由 **TM5、TM6 的胞内段及 ICL2** 构成核心腔隙，**ICL3** 在某些受体中进一步增强该相互作用（Rasmussen et al., *Nature* 2011; 后续 cryo-EM 结构）。

### 1.2 现有方法的两大核心局限

| 局限                     | 表现                                                                       | 后果                                                              |
| ------------------------ | -------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| **单蛋白分类退化** | 大多数方法将任务视为"对 GPCR 分类"，使用固定的 G 蛋白模板（如人源 GNAQ）。 | 模型从未学习成对相互作用规则；退化为仅依赖 GPCR 的分类器。        |
| **缺乏拓扑引导**   | 使用全序列平均池化嵌入，未区分结合区与非结合区。                           | 模型注意力漂移至非功能区域（N 端/C 端），遗漏真正的 ICL2/3 界面。 |

### 1.3 我们的三项解决方案

1. **真正的成对任务构建** — 每个样本均为 *(GPCR, G 蛋白家族)* 对。同一 GPCR 与不同家族配对出现，迫使模型利用 G 蛋白侧信息。
2. **拓扑感知特征工程** — 我们从精选的 UniProt 跨膜注释中提取精确的 ICL2/3 边界，注入这一结构先验知识。
3. **严格的同源控制评估** — 三级交叉验证方案（随机 → 聚类感知 → LOGPSO），以消除序列泄露。

## 第二部分：数据与方法（简述）

### 2.1 数据集

- **来源：** GPCRdb/IUPHAR 偶联注释 + 51 对 Gq 特异性本地种子数据。
- **最终可用对：** 1,639
- **唯一 GPCR：** 448（在 30% 同一性下聚类为 387 个同源组）
- **阳性：** 417（25.4%） | **阴性：** 1,222（74.6%）

### 2.2 特征

- **ESM-2 嵌入：** 主实验使用 `esm2_t6_8M_UR50D`（320 维）。我们还使用 `esm2_t33_650M_UR50D`（1,280 维）开展了平行研究，以测试模型规模效应。
- **UniProt 拓扑：** 利用 7 次跨膜螺旋定位 ICL2 和 ICL3 边界（成功率 98.8%）。
- **ICL2/3 描述符：** 每个环的 8 个理化统计量（长度、疏水性、净电荷、芳香族比例等）。
- **V2 增强（新增）：**
  - **家族级 Gq 阳性率先验** — 训练数据中某 GPCR 家族与 Gq 偶联的频率。
  - **ICL2/3 同源相似性** — ICL 统计量与最近人源直系同源物的余弦相似度。

### 2.3 三级评估

| 层级                  | 划分规则                      | 解释                     |
| --------------------- | ----------------------------- | ------------------------ |
| **随机 CV**     | 标准 5 折分层交叉验证         | 乐观上限                 |
| **聚类感知 CV** | 按同源聚类划分（CD-HIT ~30%） | 对未知受体的现实泛化     |
| **LOGPSO**      | 留一 G 蛋白家族交叉验证       | 跨家族迁移的极端压力测试 |

## 第三部分：核心实验结果

### 3.1 特征消融（主表）

**V1（原始特征，8M ESM-2）：**

| 配置         | 维度  | 随机 CV AUC    | 聚类 CV AUC              | LOGPSO AUC |
| ------------ | ----- | -------------- | ------------------------ | ---------- |
| 基线         | 640   | 0.815 ± 0.031 | 0.797 ± 0.022           | 0.638      |
| + ICL 统计量 | 656   | 0.832 ± 0.017 | 0.810 ± 0.030           | 0.628      |
| + ICL 完整   | 1,296 | 0.839 ± 0.014 | **0.830 ± 0.041** | 0.609      |
| 仅 ICL       | 656   | 0.598 ± 0.025 | 0.683 ± 0.035           | 0.551      |

**V2（含家族先验 + ICL 相似性，8M ESM-2）：**

| 配置            | 维度  | 随机 CV AUC              | 聚类 CV AUC              | LOGPSO AUC |
| --------------- | ----- | ------------------------ | ------------------------ | ---------- |
| 基线            | 640   | 0.815 ± 0.031           | 0.798 ± 0.014           | 0.638      |
| + ICL 统计量 V2 | 658   | **0.848 ± 0.020** | **0.836 ± 0.019** | 0.631      |
| + ICL 完整 V2   | 1,298 | 0.848 ± 0.015           | **0.842 ± 0.005** | 0.606      |

**V2（含家族先验 + ICL 相似性，650M ESM-2）：**

| 配置            | 维度  | 随机 CV AUC    | 聚类 CV AUC              | LOGPSO AUC |
| --------------- | ----- | -------------- | ------------------------ | ---------- |
| 基线            | 1,600 | 0.820 ± 0.033 | 0.819 ± 0.017           | 0.606      |
| + ICL 统计量 V2 | 1,618 | 0.830 ± 0.032 | **0.833 ± 0.014** | 0.601      |
| + ICL 完整 V2   | 4,178 | 0.822 ± 0.028 | 0.832 ± 0.019           | 0.593      |

**关键发现：**

- 拓扑感知特征将聚类 CV AUC 从 **0.797 提升至 0.830**（V1，+0.033）。
- 添加家族先验和 ICL 相似性后，8M 模型达到 **0.842**（总计 +0.045），650M 模型达到 **0.833**（+0.014）。
- **非单调的模型规模效应：** 650M 基线已很强（0.819），因此显式先验的*相对*增益小于 8M 模型。这表明更大的嵌入已经隐式编码了部分家族/物种信息。
- 仅 ICL 模型在随机 CV 中仅达 0.598，证实**局部拓扑与全局序列上下文是互补的，而非可相互替代的。**

### 3.2 LOGPSO：协议–表示不匹配

所有包含固定 G 蛋白向量的模型在 LOGPSO 下均崩溃（AUC ~0.60，精确率/召回率/F1 ≈ 0）。这**并非生物学瓶颈**，而是结构不兼容：当整个 G 蛋白家族被留出时，每个测试样本都携带*相同的未见过的常数块*，导致 RBF 核分配几乎相同的距离。仅 ICL 模型避免了这一问题并产生了非零指标，证实了上述诊断。

### 3.3 可解释性

- **置换重要性：** ICL2 芳香族比例和 ICL3 长度排名高于任何单一 ESM-2 维度，与结构生物学一致。
- **SHAP 区域注意力：** 添加拓扑特征后，模型权重从非结合区（N 端/C 端）向已知的结合界面（ICL2/ICL3）转移，与"结构注释将注意力引导至生物学相关区域"的假设一致。

## 第四部分：独立验证 — OPN4 测试集

我们预测了 9 种视蛋白序列是否能激活**人源 GNAQ**，并进行了严格的泄露检查和标签可靠性分级。

### 4.1 泄露分析

- **NP_150598.1**（人源黑视蛋白 iso1）：与 Q9UHM6 的 Jaccard = 1.000 → **泄露**
- **NP_001025186.1**（人源黑视蛋白 iso2）：与 Q9UHM6 的 Jaccard = 0.967 → **泄露**
- 其余 7 种非人源视蛋白的最大相似度为 0.07–0.17 → **干净**

### 4.2 V1 结果（原始特征）

总体准确率：44.4%；干净样本准确率：**28.6%**（2/7）。蜜蜂和蝴蝶视蛋白出现系统性假阳性（概率 0.55–0.74）。

### 4.3 V2 结果 — 8M 对比 650M

**V2（8M）：** 总体 77.8%；干净样本 **71.4%**

- 假阳性大幅下降（蜜蜂/蝴蝶概率降至 0.31–0.44）。
- **权衡：** 14.3% 的家族先验过强，将两个远缘阳性样本推至阈值以下（鱼类黑视蛋白和蜘蛛 kumopsin1）。

**V2（650M）：** 总体 88.9%；干净样本 **85.7%**

- 鱼类黑视蛋白概率：**0.852**（正确）
- 蜘蛛 kumopsin1 概率：**0.544**（正确）
- 一个新的假阴性：鱿鱼的视蛋白（0.519）。

**关键洞见：** 650M 模型更丰富的 1,280 维嵌入**抵抗了家族先验对远缘阳性样本的抑制作用**，带来了显著的跨物种改进。

### 4.4 标签可靠性问题

九个测试标签中有五个在人源 GNAQ 框架下被评为 "D — 不可靠"。蜜蜂和蝴蝶视蛋白被标记为 "No"，是基于"昆虫使用昆虫 Gq"的推断，而这**在逻辑上并不蕴含**它们无法激活人源 GNAQ。这意味着 V1 中的部分"错误"实际上可能反映的是有缺陷的基准测试，而非模型泛化能力差。

## 第五部分：三大核心结论

1. **成对框架消除了单蛋白退化。** SHAP 证实全部 320 个 G 蛋白 ESM-2 维度均有贡献，证明模型学习的是相互作用规则，而非仅进行 GPCR 分类。
2. **拓扑注释将预训练嵌入引导至生物学上合理的决策边界。** UniProt 精选的 7-TM 边界优于启发式方法（98.8% vs. 1.2% 成功率），模型注意力从非功能性尾部向 ICL2/3 界面转移。
3. **跨家族泛化需要新的相互作用表示。** LOGPSO 暴露了拼接固定向量方案中的*协议–表示不匹配*。未来工作应转向残基级交叉注意力或 AlphaFold 衍生的界面接触特征。

## 第六部分：局限性与未来方向

| 局限性                         | 影响 | 缓解措施 / 下一步                                          |
| ------------------------------ | ---- | ---------------------------------------------------------- |
| 缺乏 3D 结构特征               | 高   | 整合 AlphaFold 界面接触预测                                |
| LOGPSO 协议–表示不匹配        | 中   | 采用基于注意力的共嵌入                                     |
| 二值家族标签（无亚型）         | 中   | 向亚型级预测推进                                           |
| 无湿实验验证                   | 高   | 定点突变 + BRET/Co-IP                                      |
| 跨物种基准不可靠               | 高   | 将未验证样本标记为**UNKNOWN** 并将其从定量评估中移除 |
| 家族先验可能抑制远缘阳性（8M） | 中   | 使用更大的预训练模型（650M）或扩展训练集的物种覆盖度       |

---

## 结语句

**通过将 UniProt 精选的跨膜拓扑注入 ESM-2 嵌入，并引入家族级偶联先验以及 ICL2/3 同源相似性，我们将聚类 CV AUC 从 0.797 提升至 0.842（8M）和 0.833（650M），可解释性分析证实注意力向真正的结合界面转移。在独立的 OPN4 测试中，干净样本准确率从 28.6% 跃升至 71.4%（8M）和 85.7%（650M），然而人源 GNAQ 框架中对远缘物种的不可靠标注仍然是跨物种评估的关键瓶颈。**
