## 二、图表标题规范

### 图标题 (Figure Captions)

**格式规范**:
- 名词性短语: 使用 **Title Case** (所有实词首字母大写)，末尾**不加句号**
- 完整句子: 使用 **Sentence case** (仅首词大写)，末尾**必须加句号**
- 去除冗余: 去掉 "The figure shows" 或 "This diagram illustrates"，直接描述内容

**推荐标题**:

| 图表 | 中文描述 | 英文标题 (规范) |
|------|---------|----------------|
| Figure 1 | 方法流程图，展示从数据收集到结果分析的完整流程 | **Architecture of the Proposed GPCR-Gαq Coupling Prediction Framework** |
| Figure 2 | 五种方法的性能对比 | **Performance Comparison of Different Machine Learning Methods** |
| Figure 3 | ROC曲线对比 | **ROC Curves for GPCR-Gαq Coupling Prediction** |
| Figure 4 | 消融实验结果 | **Ablation Study on Feature Contributions** |
| Figure 5 | 性能热力图 | **Comprehensive Performance Metrics Across Methods** |

**完整图注示例**:
```
Figure 1. Architecture of the Proposed GPCR-Gαq Coupling Prediction Framework.
The pipeline consists of six stages: (1) Data collection from UniProt, (2) ESM-2 
feature extraction generating 320-dimensional embeddings, (3) Cross-attention 
mechanism modeling GPCR-G protein interactions, (4) 5-fold stratified cross-validation, 
(5) Performance evaluation using AUC-ROC and F1-score, and (6) Statistical 
significance testing.
```

### 表标题 (Table Captions)

**格式规范**:
- 使用 **Title Case** 格式
- 推荐句式: "Comparison with", "Ablation study on", "Results on"
- 避免: showcase, depict → 使用 show, compare, present

**推荐标题**:

| 表格 | 中文描述 | 英文标题 (规范) |
|------|---------|----------------|
| Table 1 | 数据集组成 | **Dataset Composition and G Protein Coupling Distribution** |
| Table 2 | 方法性能对比 | **Comparison of Machine Learning Methods for GPCR-Gαq Coupling Prediction** |
| Table 3 | 消融实验结果 | **Ablation Study on Different Feature Combinations** |
| Table 4 | 统计显著性检验 | **Statistical Significance Tests Between Methods** |

**完整表注示例**:
```
Table 2. Comparison of Machine Learning Methods for GPCR-Gαq Coupling Prediction.
The best results for each metric are highlighted in bold. Standard deviations 
are computed from 5-fold cross-validation.
```
