# GPCR-Gα 蛋白耦合特异性预测：Phase 2 实验结果与湿实验候选报告

**汇报日期：** 2026-04-16  
**汇报人：** 计算生物学课题组  
**面向对象：** 生物老师 / 湿实验合作团队

---

## 摘要

本研究旨在利用计算模型预测 G 蛋白偶联受体（GPCR）与 Gα 蛋白亚型（Gq、Gi、Gs、G12/13）之间的耦合特异性。在 Phase 2 补强实验中，我们系统比较了不同蛋白质语言模型（ESM-2 8M vs. 650M 参数）以及传统机器学习（SVM）与深度学习（Cross-Attention）架构的预测性能，并筛选出一批具有湿实验验证价值的候选样本。

**核心成果：**
- **最佳模型**（Cross-Attention + 650M 全局特征 + 1280-d ICL 局部特征）在严格的聚类交叉验证中达到 **AUC = 0.8599**，为目前项目组最高水平。
- 从最佳模型中筛选出 **10 个湿实验候选样本**，涵盖阳性对照、阴性对照、边界案例及争议样本，可直接用于后续的细胞实验验证。

---

## 1. 研究背景与科学问题

### 1.1 GPCR 信号通路的重要性
GPCR 是人体最大的膜受体家族，参与调控视觉、嗅觉、神经递质传递、激素响应等众多生理过程。GPCR 被配体激活后，通过与下游 **Gα 蛋白** 耦合来启动细胞内信号级联反应。不同 GPCR 对 Gα 亚型的选择性直接决定了其下游信号通路（例如 Gq 激活 PLCβ，Gi 抑制腺苷酸环化酶，Gs 激活腺苷酸环化酶）。

### 1.2 为什么要做预测？
传统上，GPCR-Gα 耦合特异性主要通过昂贵的生化实验（如 [³⁵S]GTPγS 结合实验、BRET、IP₁ 积累测定）来确定。对于数百种孤儿 GPCR 或工程改造受体，实验通量极低。因此，建立高精度的**计算预测模型**可以：
- 为湿实验提供**优先级排序**，减少盲目筛选；
- 揭示决定耦合特异性的**关键序列区域**（如 ICL2、ICL3）；
- 指导 GPCR 定向进化或药物设计。

## 2. 实验设计与方法（简要版）

### 2.1 数据集
- **样本量：** 1,639 对 GPCR-Gα 配对记录（正样本 = 已知耦合，负样本 = 无耦合记录）。
- **序列聚类：** 387 个序列同源簇（cluster）。为避免模型"记住"与测试样本相似的序列，我们采用了**聚类感知交叉验证**（Cluster-aware CV），确保训练集与测试集在序列上无同源关系，这是比随机划分更严格的评估标准。

### 2.2 模型与特征
我们将问题抽象为"给定一条 GPCR 序列和一种 Gα 亚型，预测它们是否耦合"。使用了两种计算框架：

1. **支持向量机（SVM）**：经典的非线性分类器，作为稳健基线。
2. **Cross-Attention 神经网络**：让 GPCR 表征"主动关注"Gα 蛋白的表征，模拟两者相互作用时的信息交换，是近年来蛋白-蛋白相互作用预测的主流架构。

**输入特征**来源于蛋白质语言模型 **ESM-2**：
- **8M 模型：** 320 维向量，轻量、快速；
- **650M 模型：** 1,280 维向量，包含更丰富的进化与结构信息；
- **ICL 局部特征：** ICL2（第3与第4跨膜区之间）和 ICL3（第5与第6跨膜区之间）是 GPCR 与 Gα 直接接触的界面区域。我们额外提取了这两个区域的 ESM-2 特征和理化统计量（长度、疏水性、电荷等）。

### 2.3 评估指标
采用 **AUC（ROC 曲线下面积）** 作为主要指标，衡量模型区分正负样本的综合能力。AUC = 0.5 相当于随机猜测，AUC 越接近 1.0 表示预测能力越强。

## 3. 主要实验结果

### 3.1 性能对比总览

下表展示了不同方法在严格聚类交叉验证下的 AUC 表现：

| 方法 | 蛋白质语言模型 | 是否加入 ICL 局部特征 | Cluster-aware AUC |
|------|----------------|----------------------|-------------------|
| SVM 基线 | ESM-2 8M (320-d) | 否 | 0.7972 |
| SVM 增强 | ESM-2 8M (320-d) | **是** | **0.8301** |
| SVM 基线 | ESM-2 650M (1280-d) | 否 | 0.8131 |
| SVM 增强 | ESM-2 650M (1280-d) | **是（1280-d ICL）** | **0.8304** |
| Cross-Attention 基线 | ESM-2 8M (320-d) | 否 | 0.8127 |
| Cross-Attention 增强 | ESM-2 8M (320-d) | **是** | 0.8223 |
| Cross-Attention 基线 | ESM-2 650M (1280-d) | 否 | 0.8393 |
| **Cross-Attention 增强** | **ESM-2 650M (1280-d)** | **是（1280-d ICL）** | **0.8599** |

*可视化对比图见 `results_comparison.png`*

### 3.2 结果解读

**（1）大模型预训练特征带来稳定提升**
无论是 SVM 还是 Cross-Attention，将全局特征从 320-d（8M）升级到 1280-d（650M）后，基线性能均有提升。这说明更强大的蛋白质语言模型能捕捉到更多与 GPCR-Gα 耦合相关的进化与结构信号。

**（2）传统机器学习方法存在性能天花板**
SVM 在 8M 模型下达到 0.8301，在 650M 模型下仅微增至 0.8304。这表明对于当前数据规模和特征形式，SVM 的 RBF 核函数已接近其表达能力上限，难以进一步挖掘 650M 高维特征中的深层模式。

**（3）深度学习架构能充分利用高维特征**
Cross-Attention 在 650M + ICL 配置下达到 **0.8599**，不仅显著优于同配置 SVM（+2.95%），也较自身 8M 版本提升 3.76%。这证明复杂的神经网络架构（特别是模拟蛋白-蛋白相互作用的 Cross-Attention 机制）对高质量预训练表征有更好的利用能力。

**（4）局部 ICL 特征必须与全局特征维度匹配**
一个意外的发现是：当我们用 1280-d 全局特征搭配旧版 320-d ICL 特征时，Cross-Attention 的性能反而下降至 0.8433；而在将 ICL 特征也重新提取为 **1280-d** 后，性能才跃升至 0.8599。

**生物学启示：** ICL2/ICL3 作为耦合界面的局部特征，只有当它们与全局序列特征处于同一"表征空间"时，才能被模型有效整合。异构维度会引入噪声，干扰耦合界面的正确识别。

## 4. 湿实验候选样本筛选

基于性能最佳的 **Cross-Attention + 650M + 1280-d ICL** 模型，我们对全部 1,639 对样本进行了全数据训练与预测概率输出，并从中科学性地筛选出 **10 个湿实验候选**。

筛选逻辑如下：
- **高置信阳性：** 包含已知阳性对照（验证实验体系可靠性）+ 模型最有信心的新预测（novel prediction）。
- **中置信：** 预测概率最接近 0.50 的样本，处于决策边界，最能检验模型的分辨极限，也常对应生物学上"弱耦合"或"条件依赖性耦合"的真实案例。
- **高置信阴性：** 模型强烈预测为不耦合的样本，作为阴性对照。
- **争议样本：** 模型"最自信的错误"——高置信假阳性和低概率假阴性，它们往往提示数据标签噪声、条件依赖性耦合或模型尚未掌握的物理化学规则。

| 类别 | GPCR ID | Gα 亚型 | 已知标签 | 预测概率 | 科学意义 |
|------|---------|---------|----------|----------|----------|
| **高置信阳性** | 5_P28335 | **Gq** | 已知耦合 (1) | **1.0000** | 阳性对照，验证实验体系 |

### 湿实验验证建议
1. **优先验证 G_O43603 (Gq)**：这是模型推荐的 top novel 预测，若实验阳性，将直接证明模型的发现能力。
2. **重点检验 O75084 (Gs)**：该样本在数据库中标记为 Gs 耦合，但模型给出极低概率。实验验证可澄清是模型缺陷还是数据库标签/条件特殊性导致。
3. **中置信 trio (O15529, O00144, Q9Y2T6)**：建议通过 BRET 或 GloSensor 在不同配体刺激下测试，评估是否存在配体依赖的弱耦合现象。

## 5. 关键发现与生物学意义总结

1. **预训练大模型对膜蛋白-蛋白相互作用预测具有显著增益。** 650M 参数的 ESM-2 相比 8M 版本，使最佳模型 AUC 提升近 3.8%，这对于湿实验筛选的阳性命中率有直接价值。
2. **深度学习的 Cross-Attention 架构优于传统机器学习。** 它能模拟 GPCR 与 Gα 之间的"注意力焦点"，捕捉到 SVM 无法建模的非线性互作模式。
3. **ICL2/ICL3 局部特征必须与全局特征处于同一表征空间。** 这从计算方法学角度强调了耦合界面的局部进化信号需要与受体整体背景信号对齐，才能被模型正确解读。
4. **SVM 在该任务上存在 ~0.83 的性能天花板。** 若未来数据量不再大幅增加，建议以深度学习（Cross-Attention 或其变体）作为主力模型框架。

## 6. 下一步计划

### 近期（1–2 个月）
- **湿实验验证：** 委托生物实验室对 `wetlab_candidates_650m.json` 中的 10 个候选进行 [³⁵S]GTPγS 或 BRET 耦合实验。
- **结果反馈：** 将实验结果与模型预测对比，计算阳性预测率（PPV）和假阳性率（FPR），评估 0.8599 AUC 在实际湿实验中的转化率。

### 中期（3–6 个月）
- **Attention 可视化：** 解析 Cross-Attention 的权重矩阵，定位 GPCR 序列中对 Gα 选择性最关键的氨基酸残基，为定点突变实验提供指导。
- **拓展至 orphan GPCR：** 将训练好的 650M Cross-Attention 模型应用于 Class A 孤儿受体，生成大规模 Gα 耦合预测谱。

### 长期
- 若有更大规模 ESM 模型（如 ESM-2 3B）或 AlphaFold-Multimer 结构信息，可进一步融合结构对接分数，尝试突破 0.86 的天花板。
