## 四、关键发现

### 1. 传统方法优于深度学习（小样本场景）

在小样本(47个)情况下：
- SVM (AUC=96.39%) > 简化版交叉注意力 (AUC=82.28%)
- 说明：深度学习需要更多数据才能发挥优势

### 2. ESM-2特征强大

仅使用ESM-2嵌入(320维)就能达到96%+的AUC：
- 预训练蛋白质语言模型的特征提取能力优秀
- 无需复杂特征工程

### 3. 样本不平衡问题

- 正样本(29)多于负样本(18)
- 需要补充更多非Gq偶联的GPCR作为负样本

### 4. 召回率100%

所有SVM模型在测试集上召回率达到100%：
- 无假阴性（没有漏检）
- 对正样本识别能力强
