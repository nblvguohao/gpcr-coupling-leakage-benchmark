## 主要发现

1. **传统方法优于深度学习**: 在小样本(47个)情况下，SVM (AUC=96.39%) 优于简化版交叉注意力模型 (AUC=82.28%)

2. **ESM-2特征强大**: 仅使用ESM-2嵌入(320维)就能达到96%+的AUC，说明预训练蛋白质语言模型的特征提取能力

3. **样本不平衡**: 正样本(29)多于负样本(18)，需要更多负样本

4. **召回率100%**: 所有SVM模型在测试集上召回率达到100%，没有假阴性
