## 一、P0级任务（必须完成）

### 任务1: 数据扩展至100+样本

**当前状态**: 45个样本  
**目标**: 100+样本  
**缺口**: 55+样本  
**时间**: 2周  

#### 具体执行方案

**阶段1A: GPCRdb数据库挖掘（3天）**

1. 访问GPCRdb (https://gpcrdb.org/)
2. 下载G蛋白偶联数据
3. 筛选Gq/11偶联的GPCR
4. 提取序列和标签

**预期收获**: 30-40个新样本

**验收标准**:
- [ ] 从GPCRdb获得≥30个Gq偶联GPCR序列
- [ ] 包含至少10个不同物种
- [ ] 数据格式统一（FASTA + CSV标签）

---

**阶段1B: 文献挖掘（4天）**

1. PubMed搜索关键词:
   - "opsin Gq coupling"
   - "melanopsin G protein"
   - "GPCR Gq/11 activation"
   - "opsin signal transduction"

2. 筛选标准:
   - 有实验验证的G蛋白偶联数据
   - 提供蛋白质序列或NCBI ID
   - 发表在高影响力期刊

3. 提取信息:
   - 蛋白质名称和序列
   - 物种来源
   - 实验验证方法（BRET, Co-IP, 信号通路分析）
   - G蛋白偶联类型

**预期收获**: 20-30个新样本

**验收标准**:
- [ ] 从文献获得≥20个实验验证样本
- [ ] 包含文献引用（PMID）
- [ ] 有明确的实验验证信息

---

**阶段1C: 同源蛋白扩展（3天）**

1. 使用NCBI BLAST搜索同源序列
2. 筛选标准:
   - 序列相似性: 60-90%
   - 覆盖率: >80%
   - E-value: <1e-5

3. 使用OrthoDB查找直系同源蛋白

**预期收获**: 10-15个新样本

**验收标准**:
- [ ] 获得≥10个同源蛋白序列
- [ ] 提供相似性分数和E-value
- [ ] 确保功能保守性

---

**阶段1D: 负样本生成（2天）**

1. 选择已知不偶联Gq的GPCR:
   - Gi/o偶联GPCR
   - Gs偶联GPCR
   - G12/13偶联GPCR

2. 从GPCRdb获取序列

3. 确保负样本多样性:
   - 不同家族（Class A, B, C）
   - 不同物种
   - 不同功能类型

**预期收获**: 15-20个负样本

**验收标准**:
- [ ] 获得≥15个高质量负样本
- [ ] 包含不同G蛋白偶联类型
- [ ] 与正样本有合理的序列相似性

---

#### 数据质量控制

**检查清单**:
- [ ] 所有序列长度在300-600 aa之间
- [ ] 无重复序列（相似性<95%）
- [ ] 标签准确（有文献或数据库支持）
- [ ] 物种多样性（至少20个不同物种）
- [ ] 序列格式正确（标准氨基酸字母）

**数据验证**:
- 使用CD-HIT去冗余（阈值95%）
- 使用InterProScan验证结构域
- 手动检查10%的随机样本

---

### 任务3: 实验验证

**当前状态**: 无实验验证  
**目标**: 至少3个预测的实验验证  
**时间**: 2周（取决于合作者响应速度）  

#### 具体执行方案

**阶段3A: 文献验证（3天）**

1. 选择3-5个模型预测的高置信度样本
2. 搜索这些蛋白质的实验验证文献
3. 提取实验数据:
   - 实验方法（BRET, Co-IP, 信号通路分析）
   - 偶联效率（EC50, 最大响应）
   - 实验条件（细胞系，表达水平）

**验收标准**:
- [ ] 找到≥3个预测的实验证据
- [ ] 实验数据与预测一致
- [ ] 提供完整的文献引用

---

**阶段3B: 联系实验合作者（5天）**

1. **识别潜在合作者**:
   - 搜索发表过opsin/GPCR实验研究的实验室
   - 重点关注有BRET或Co-IP经验的团队
   - 优先选择有开放合作历史的团队

2. **准备合作材料**:
   - 研究摘要（1页）
   - 预测结果列表（top 10）
   - 实验验证方案草案
   - 预期合作模式（数据共享，共同发表）

3. **发送合作邀请**:
   - 个性化邮件
   - 突出研究价值
   - 明确合作贡献

**验收标准**:
- [ ] 联系≥5个潜在合作者
- [ ] 收到≥2个积极回应
- [ ] 确定≥1个合作意向

---

**阶段3C: 公开数据集验证（2天）**

1. 搜索公开的GPCR-G蛋白相互作用数据集
2. 下载验证数据
3. 对比模型预测与实验数据

**潜在数据源**:
- GPCRdb实验数据

**验收标准**:
- [ ] 找到≥1个公开验证数据集

## 五、风险与应对

| 风险 | 可能性 | 影响 | 应对策略 |
|------|--------|------|---------|
| GPCRdb数据不足 | 低 | 高 | 使用多个数据库，降低筛选标准 |
| AlphaFold预测失败 | 低 | 中 | 使用ESMFold，或跳过结构特征 |
| 实验合作者无回应 | 中 | 高 | 多联系几个团队，使用文献验证替代 |
| 时间超期 | 中 | 中 | 优先完成P0任务，P1任务简化 |
| 性能提升不明显 | 低 | 高 | 增加数据增强，尝试集成学习 |
